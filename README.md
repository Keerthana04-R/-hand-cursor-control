# Hand Cursor Control 🖐️

Control your computer cursor using hand gestures and AI — no mouse needed!

## Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- NumPy

## Gestures
| Gesture | Fingers | Action |
|---|---|---|
| ☝️ One finger | Index only | Move cursor |
| 🤏 Pinch | Thumb + Index | Left click |
| 🤌 Pinch middle | Thumb + Middle | Right click |
| ✌️ Two fingers | Index + Middle | Scroll up |
| 🖖 Three fingers | Index + Middle + Ring | Scroll down |
| 🖐️ Four fingers | All four | Screenshot |

## Installation
pip install opencv-python mediapipe pyautogui numpy

## How to Run
python hand_cursor.py
